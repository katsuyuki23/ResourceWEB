# buat-dashboard-website-berbasi

Buat dashboard website berbasis HTML, CSS, JavaScript dengan konsep Islamic Modern, clean, profesional, dan responsif untuk manajemen panti asuhan, bersifat admin-only system. Sistem Login hanya untuk Admin (role-based) dengan session login wajib untuk mengakses semua halaman. Struktur dashboard menggunakan sidebar navigasi modern dengan fitur collapse/expand, ikon responsif, dan animasi smooth. Warna tema Islamic modern dengan sidebar menu: Dashboard, Kelola User, Kelola Donatur, Data Anak, Input Donasi, Input Pengeluaran, Laporan Keuangan, Berita/Pengumuman, Logout. Struktur data meliputi USERS, ROLES, DONATUR, DONASI, PENGELUARAN, ANAK. Fitur setiap halaman: Dashboard (ringkasan total donasi, total pengeluaran, jumlah anak, grafik keuangan, notifikasi singkat), Kelola User (CRUD user, assign role, reset password), Kelola Donatur (tabel donatur, tambah/edit/hapus, search & filter), Data Anak (tabel anak panti, detail biodata, status aktif/tidak, upload foto), Input Donasi (form input donasi, otomatis masuk ke laporan, relasi ke donatur), Input Pengeluaran (form pengeluaran, kategori & tanggal, tercatat otomatis ke laporan), Laporan Keuangan (rekap pemasukan & pengeluaran, filter per tanggal, export PDF/Excel), Berita/Informasi (input berita kegiatan panti, tampil di semua halaman dashboard). Desain & UX: tema Islamic modern minimalist, animasi transisi halus, icon islami, font elegan, background soft. Teknologi: HTML5, CSS3, JavaScript, Bootstrap/Tailwind, Chart.js, FontAwesome/Lucide Icons, Laravel/Node.js backend.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to the URL shown in the terminal.
